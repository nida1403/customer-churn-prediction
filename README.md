# Customer Churn Prediction

This project predicts customer churn using machine learning techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Results
- Accuracy: 94.6%
- ROC-AUC: 0.93

## Key Features
- Data preprocessing
- Feature encoding
- Random Forest model
- Performance evaluation

## Business Value
This model helps identify customers at risk of leaving, enabling proactive retention strategies.
